# 🚀 Vue 3 Login Bootstrap Project

A modern **Vue 3 + Bootstrap 5 Login UI system** built for learning frontend structure, components, and simple authentication flow.

---

## 📦 Tech Stack

- ⚙️ 0 (Vue 3 Composition API)
- 🎨 1 (UI Styling)
- ⚡ Vite (Fast build tool)

---

## ✨ Features

- 🔐 Simple login UI (email/password)
- 🎨 Clean Bootstrap 5 design
- 📱 Fully responsive layout
- 🧭 Dashboard page after login
- 🧩 Component-based architecture
- ⚡ Fast Vue 3 reactivity
